# 视频生成操作规则

## 核心原则
每次进行视频生成操作时，必须完整执行以下流程，确保留下完整的操作记录素材。

## 操作前准备

### 1. 开启录屏
- **软件**：Xbox Game Bar（Win + Alt + R 快捷键）
- **要求**：全程录屏，不得中断
- **保存位置**：项目/视频内容/对应项目/素材/

### 2. 语音解释
- **时机**：每一步操作前
- **内容**：说明当前步骤的目的和操作方法
- **示例**：
  - "大家好，我是傻小福，现在我们开始生成视频"
  - "首先，我需要在提示词输入框中输入视频描述"
  - "然后，选择视频模型为Seedance 2.0"
  - "最后，点击生成按钮"

### 3. 添加字幕
- **软件**：Xbox Game Bar 小须功能 或 剪映
- **要求**：字幕与语音同步
- **内容**：包含操作步骤说明

## 浏览器操作流程

### 步骤一：打开AI生成平台https://jimeng.jianying.com/ai-tool/generate/?type=video
- 语音："现在打开即梦AI视频生成页面"
- 字幕：打开即梦AI

### 步骤二：检查脚本并上传参考素材
- 语音："根据脚本内容检查涉及的人物和商标，然后上传对应的参考素材"
- **脚本检查与上传规则**：
  - **语言一致性**：脚本是中文→参考图描述用中文；脚本是英文→参考图描述用英文
  - 如果涉及**达程/Hugh** → 在输入框输入"@" → 弹出图片选择 → 选择达程图片 → 系统显示"图片1" → 选择达程音频 → 系统显示"音频1" → 在输入框添加"图片1是达程，音频1是达程声音"（中文）或 "图片1 is Hugh, 音频1 is Hugh's voice"（英文）
  - 如果涉及**晓舒/Shu** → 在输入框输入"@" → 弹出图片选择 → 选择晓舒图片 → 系统显示"图片2" → 选择晓舒音频 → 系统显示"音频2" → 在输入框添加"图片2是晓舒，音频2是晓舒声音"（中文）或 "图片2 is Shu, 音频2 is Shu's voice"（英文）
  - 如果涉及**SillyMD/挺傻的** → 在输入框输入"@" → 弹出图片选择 → 选择SillyMD商标图片 → 系统显示"图片3" → 在输入框添加"图片3是SillyMD商标"（中文）或 "图片3 is SillyMD logo"（英文）
  - 如果涉及**一块变/One** → 在输入框输入"@" → 弹出图片选择 → 选择一块变商标图片 → 系统显示"图片4" → 在输入框添加"图片4是一块变商标"（中文）或 "图片4 is One logo"（英文）
- **重要**：@是触发图片选择，不是直接输入文本
- 字幕：检查脚本，通过@选择对应参考素材，添加说明文字

### 步骤三：填写人物描述
- 语音："在提示词框内先填写必须的人物描述，包括达程形象和晓舒形象"
- 人物描述模板：
  - 【达程形象】37岁，皮肤状态：电影感广告质感，4k高清，商业胶片质感...
  - 【晓舒形象】30岁，皮肤质感：电影感广告片标准...
  - 【技术】画质：4K，24fps，商业胶片质感...
- 字幕：填写人物描述和技术参数

### 步骤四：输入提示词
- 语音："最后填写视频场景的详细描述，按照剧情需要"
- 字幕：输入视频提示词

### 步骤五：选择模型和参数
- 语音："选择Seedance 2.0模型，时长15秒，全能参考模式，画面比例9:16"
- 字幕：设置生成参数

### 步骤六：点击生成
- 语音："所有设置已完成，点击生成按钮"
- 字幕：点击生成

### 步骤七：等待生成
- 语音："视频正在生成中，请耐心等待"
- 字幕：等待生成

### 步骤八：下载保存
- 语音："视频生成完成，现在下载保存到本地项目文件夹"
- 字幕：下载保存

## 台词与字幕规则

### 人物中英文名对照
| 中文名 | 英文名 |
|--------|--------|
| 达程 | Hugh |
| 晓舒 | Shu |
| 傻小福 | Silly |
| 一块变 | One |
| 挺傻的 | SillyMD |

### 品牌名称
- **SillyMD**：英文名，中文名"挺傻的"
- **一块变**：中文名，英文名"One"

### 参考图说明
- 翻译成英文时，参考图说明也用英文
- 例如：SillyMD logo, One logo

### 平台规则处理
- 如果提示"不符合平台规则，请修改后重试"
- **解决方案**：将中文剧情描述翻译成英文后再试

### 字幕要求
- **格式**：中英双语字幕
- **位置**：画面底部居中
- **样式**：白字黑边，便于识别
- **示例**：
  - 中文字幕：大家好，我是达程
  - 英文字幕：Hello everyone, I'm Hugh

### 声道设定
- **语言**：中文普通话 + 英语
- **声道**：双声道（左声道中文，右声道英文）
- **技术参数**：AAC编码，44.1kHz，立体声

### 人物描述模板（含英文）
- 【达程形象/Hugh】37岁，皮肤状态：电影感广告质感，4k高清，商业胶片质感...
- 【晓舒形象/Shu】30岁，皮肤质感：电影感广告片标准...
- 【技术】画质：4K，24fps，商业胶片质感，电影感广告片风格，真实感：生活痕迹。

## 素材命名规则
- 格式：操作描述_日期时间.mp4
- 示例：大阅兵_升旗仪式_20260219.mp4

## 注意事项
1. 录屏必须完整，中途不得停止
2. 语音解释要清晰响亮
3. 字幕要与语音同步
4. 操作每一步都要说明目的
