## 🔧 开发环境要求

### Android开发
- JDK 8 或 JDK 11
- Android Studio Hedgehog (2023.1.1)+
- Gradle 8.0+

### 后端开发
- Python 3.8+
- MySQL 8.0+
- pip包管理器

### 构建命令
```bash
# Android
cd AIActive/android
gradlew.bat clean
gradlew.bat assembleDebug

# 后端
cd AIActive/backend
pip install -r requirements.txt
python app.py
```

## ⚠️ 注意事项

1. **敏感配置**: 项目包含配置文件模板，需要填入实际的API密钥
2. **离线支持**: 核心功能支持离线运行，但AI生成需要网络
3. **数据库初始化**: 需要运行 `schema.sql` 和 `init_data.sql`
4. **安全建议**: 
   - 生产环境强制HTTPS
   - 所有密钥存储在配置文件
   - 参数化查询防SQL注入
