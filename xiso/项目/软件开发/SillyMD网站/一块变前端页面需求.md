## 一、项目概述

### 1.1 项目背景
一块变是春节要上线的AI图片生成服务微信小程序，用户可以通过微信支付充值获得AI生成次数，将普通照片转换为各种艺术风格。

### 1.2 核心流程
```
点击"按一下，一块变"按钮
↓
校验用户身份（微信登录）
↓
检查账户成长分
↓
[成长分不足] → 付款页面（微信支付）
↓
[成长分充足] → 转发裂变引导
↓
上传照片
↓
展示AI生成结果
```

### 1.3 技术栈
- **前端框架**: 微信小程序原生（TypeScript）
- **组件框架**: glass-easel
- **状态管理**: 小程序 Page/Component data
- **UI组件**: 自定义组件库
- **API通信**: 封装的 HTTP 工具类

## 四、技术要点

### 4.1 微信登录流程
```typescript
// 登录流程
async function wxLogin() {
  // 1. 获取微信code
  const { code } = await wx.login();
  
  // 2. 调用后端接口
  const res = await api.auth.login({ code });
  
  // 3. 保存token
  wx.setStorageSync('token', res.data.token);
  
  // 4. 保存用户信息
  wx.setStorageSync('userInfo', res.data.userInfo);
  
  // 5. 返回上一页
  wx.navigateBack();
}
```

### 4.2 成长分检查逻辑
```typescript
// 检查成长分
async function checkBalance(requiredPoints: number) {
  const userInfo = await api.user.getProfile();
  
  if (userInfo.balance < requiredPoints) {
    // 成长分不足，跳转充值
    wx.showModal({
      title: '成长分不足',
      content: `需要${requiredPoints}成长分，当前${userInfo.balance}`,
      confirmText: '去充值',
      success: (res) => {
        if (res.confirm) {
          wx.navigateTo({
            url: '/pages/mine/recharge'
          });
        }
      }
    });
    return false;
  }
  
  return true;
}
```

### 4.3 图片上传流程
```typescript
// 上传照片
async function uploadPhoto(tempFilePath: string, styleId: string) {
  wx.showLoading({ title: '上传中...' });
  
  const uploadTask = wx.uploadFile({
    url: apiBaseUrl + '/generation/upload',
    filePath: tempFilePath,
    name: 'photo',
    formData: {
      styleId: styleId
    },
    header: {
      'Authorization': 'Bearer ' + wx.getStorageSync('token')
    }
  });
  
  uploadTask.onProgressUpdate((res) => {
    console.log('上传进度', res.progress);
  });
  
  const res = await uploadTask;
  wx.hideLoading();
  
  return JSON.parse(res.data);
}
```

### 4.4 AI生成轮询
```typescript
// 轮询生成状态
async function pollGeneration(taskId: string): Promise<GenerationResult> {
  return new Promise((resolve, reject) => {
    const pollInterval = setInterval(async () => {
      const status = await api.generation.getStatus(taskId);
      
      if (status.status === 'completed') {
        clearInterval(pollInterval);
        resolve(status);
      } else if (status.status === 'failed') {
        clearInterval(pollInterval);
        reject(new Error('生成失败'));
      }
    }, 2000); // 每2秒轮询一次
    
    // 超时处理（5分钟）
    setTimeout(() => {
      clearInterval(pollInterval);
      reject(new Error('生成超时'));
    }, 5 * 60 * 1000);
  });
}
```

### 4.5 分享海报生成
```typescript
// Canvas绘制海报
async function generateSharePoster(userAvatar: string, qrCode: string) {
  const ctx = wx.createCanvasContext('shareCanvas');
  
  // 绘制背景
  ctx.drawImage('/images/poster-bg.png', 0, 0, 375, 667);
  
  // 绘制头像
  ctx.save();
  ctx.beginPath();
  ctx.arc(187.5, 150, 50, 0, 2 * Math.PI);
  ctx.clip();
  ctx.drawImage(userAvatar, 137.5, 100, 100, 100);
  ctx.restore();
  
  // 绘制二维码
  ctx.drawImage(qrCode, 137.5, 400, 100, 100);
  
  // 绘制文案
  ctx.setFontSize(18);
  ctx.setFillStyle('#ffffff');
  ctx.fillText('扫码一起玩AI拍照秀', 110, 550);
  
  ctx.draw();
  
  // 导出图片
  return new Promise((resolve) => {
    wx.canvasToTempFilePath({
      canvasId: 'shareCanvas',
      success: (res) => resolve(res.tempFilePath)
    });
  });
}
```
