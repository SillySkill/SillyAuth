## 🔴 高优先级 (立即修复)

### F-001: 上传后URL未保存 ✅ 已修复
- **模块**: 前端 - upload.ts
- **严重程度**: 🔴 高
- **描述**: 图片上传成功后，服务器返回的URL保存在uploadedUrls数组中，但没有使用或传递到下一页
- **影响**: 导致后续生成流程无法获取正确的图片URL
- **状态**: ✅ 已修复 (2026-02-15)
- **修复内容**:
  - 新增 serverUrls 字段保存服务器URL
  - uploadImages 方法保存服务器返回的URL
  - onNextStep 传递 serverUrls 参数到下一页

### B-001: 缩略图未压缩 ✅ 已修复

### B-004: 未验证支付签名 ✅ 已修复
- **模块**: 后端 - api_yibian.py
- **严重程度**: 🔴 高
- **描述**: 支付回调未验证微信支付签名，存在严重安全漏洞
- **影响**: 可能被伪造支付回调，造成资金损失
- **状态**: ✅ 已修复 (2026-02-15)
- **修复内容**:
  - 添加 verify_wechat_sign() 函数
  - 解析XML获取签名
  - 使用MD5验证签名

---

### B-005: 支付回调格式错误 ✅ 已修复
- **模块**: 后端 - api_yibian.py
- **严重程度**: 🔴 高
- **描述**: 接收JSON格式，但微信支付回调是XML格式
- **影响**: 无法正确接收微信支付回调
- **状态**: ✅ 已修复 (2026-02-15)
- **修复内容**:
  - 修改接口接收 Request 对象
  - 使用 await request.body() 获取原始XML
  - 使用 xml.etree.ElementTree 解析XML

## ⚠️ 部署注意事项

1. **环境变量配置**:
   - 复制 `.env.example` 为 `.env`
   - 填入真实的 API Key 和密钥
   - 生产环境必须修改 JWT_SECRET

2. **依赖安装**:
   ```bash
   pip install python-jose[cryptography]
   pip install bcrypt
   pip install Pillow
   pip install python-dotenv
   ```

3. **测试验证**:
   - 测试上传功能，验证缩略图生成
   - 测试登录功能，验证JWT Token
