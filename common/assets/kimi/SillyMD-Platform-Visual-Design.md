# SillyMD Platform 视觉设计方案

## 1. 设计理念

### 整体视觉方向

- 整体视觉将呈现一种精密、前瞻且高度结构化的科技美学。设计灵感源自《MIT Technology Review》的深度与严谨，以及电影《机械姬》中冷静、智能的未来主义氛围。核心理念是通过严谨的网格系统和克制的色彩，构建一个清晰、可信赖的视觉叙事，以匹配平台作为AI资产托管中心的技术深度与前瞻性。

### 内容密度

- 高密度。这是一个技术平台架构文档，包含大量专业术语、流程图和数据表格。设计需优先保证信息传递的清晰与高效，每页可承载3-5个核心要点，图文占比高，留白控制在15%以内，以结构化的布局呈现复杂信息。

## 2. 色彩系统

**主色调**

- 背景色：#1A1D21（深灰黑色，营造专业科技氛围）
- 主色：#5A6B7B（冷静蓝灰，代表技术理性）
- 辅助色：#8D99AE（柔和灰蓝，用于次要信息）
- 强调色：#4DB6AC（青绿色，突出关键数据和交互点）

**文字颜色**

- 深色文字（用于浅色背景）：#2B2D42
- 浅色文字（用于深色背景）：#E0E0E0

## 3. 字体设计

**标题字体**

- 中文字体
  - Chiron Hei HK（现代几何黑体，线条锐利，科技感突出，契合AI主题）
  - Noto Sans SC（思源黑体，风格中性规范，作为备选保证兼容性与可读性）
- 英文字体
  - Hedvig Letters Sans（个性鲜明的无衬线体，为技术主题注入独特的品牌气质）
  - Quattrocento Sans（经典优雅的无衬线体，结构开放，确保标题的清晰易读）

**正文字体**

- 中文字体
  - Noto Sans SC（思源黑体，中性、清晰、可读性极高，适合大段技术说明）
  - Noto Serif SC（思源宋体，为长文阅读提供另一种典雅、正式的视觉选择）
- 英文字体
  - Quattrocento Sans（经典无衬线体，温和易读，在小字号下依然表现清晰）
  - Sorts Mill Goudy（古典印刷风格衬线体，赋予文档一种权威、严谨的学术感）

## 4. 图像风格

- 图像风格应为抽象和概念化的。优先使用数据可视化图表、网络节点图、抽象的几何图形或微观的电路/代码纹理。避免使用具象的AI机器人或人物照片。整体色调应与配色系统保持一致，采用冷色调、低饱和度，营造出专业、冷静且富有深度的视觉氛围。
