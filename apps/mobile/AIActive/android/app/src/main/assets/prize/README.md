# 知识问答奖品资源目录

本目录用于存放知识问答模块的奖品图片。

## 资源文件

**默认奖品图片**: `quiz_prize.png`

这是答题达到门槛后显示的奖品图片。

## 图片规格要求

- **格式**：PNG（推荐，支持透明背景）或 JPG
- **推荐尺寸**：800 x 600 像素（4:3比例）
- **最小尺寸**：400 x 300 像素
- **文件大小**：建议 < 500KB
- **设计建议**：
  - 突出奖品形象（礼品盒、奖杯、奖牌等）
  - 色彩鲜明，吸引眼球
  - 可以添加"恭喜中奖"等文字

## 配置说明

### 1. 在设置页面配置（推荐）

进入：**系统设置** → **知识问答设置**

- **奖品门槛**：设置答对几题触发（如3题）
- **启用奖品**：开关奖品功能

### 2. 修改配置文件

编辑 `question/config.json`：

```json
"prizes": {
  "enabled": true,
  "default_threshold": 3,
  "default_prize_image": "quiz_prize.png"
}
```

## 临时方案

如果暂时没有准备奖品图片，系统会使用默认图标显示，不影响功能使用。

## 更换奖品图片

1. 将新的奖品图片命名为 `quiz_prize.png`
2. 放到本目录（`assets/prize/`）
3. 重新构建应用

## 设计建议

可以使用以下元素：
- 🎁 礼品盒
- 🏆 奖杯
- 🎖️ 奖牌
- 🎉 庆祝元素
- ✨ 闪光特效
