## 创建步骤

### 使用 Microsoft Excel

1. 打开 Microsoft Excel
2. 创建新的工作簿
3. 在 Sheet1 中输入以下列标题（第一行）：
   - A1: 候选人ID
   - B1: 姓名
   - C1: 部门
   - D1: 备注

4. 从第二行开始填入候选人数据

5. 保存为 `prizeDate.xlsx`（注意是 .xlsx 格式）

6. 复制到 `android/app/src/main/assets/lottery/` 目录

### 使用 WPS 表格

1. 打开 WPS 表格
2. 创建新的工作表
3. 按照上述格式输入数据
4. 另存为 XLSX 格式
5. 复制到 `android/app/src/main/assets/lottery/` 目录

## 数据验证规则

- ✅ 姓名不能为空（必填）
- ✅ 部门不能为空（必填）
- ✅ 候选人ID建议唯一（选填，自动生成）
- ✅ 备注可选


2. 进入"幸运抽奖"
3. 选择"礼花"
4. 查看"剩余候选人"显示

## 常见问题

**Q: 如何验证Excel文件正确？**
A: 打开 Excel 文件，确认：
- 第一行为标题
- 至少有一行数据
- 姓名列（B列）不为空

**Q: 可以有多个工作表吗？**
A: 可以，但只会读取第一个工作表（Sheet1）

**Q: 支持多少候选人？**
A: 理论上无限制，建议不超过1000人

**Q: 支持中文姓名吗？**
A: 完全支持，使用 UTF-8 编码
