## 六、种子数据管理

### 6.1 清理种子数据

**一键清理所有种子数据：**
```sql
SELECT clean_seed_data();
```

该函数会：
1. 删除所有种子 Skills 相关数据（评论、收藏、标签、审核等）
2. 删除所有种子用户相关数据（偏好、通知、交易记录等）
3. 删除映射表
4. 返回删除统计信息

### 6.2 查看种子数据详情

**种子用户详情：**
```sql
SELECT
    u.username,
    u.email,
    u.role,
    u.vendor_level,
    u.bio,
    u.created_at
FROM users u
WHERE u.email LIKE '%@seed.local';
```

**种子 Skills 详情：**
```sql
SELECT
    s.skill_id,
    s.name,
    s.type,
    s.price,
    s.rating_avg,
    s.download_count,
    u.username as author_name
FROM skills s
JOIN users u ON s.author_id = u.id
WHERE s.skill_id LIKE 'seed_%';
```

### 8.1 数据统计查询

```sql
SELECT
    role,
    vendor_level,
    COUNT(*) as count
FROM users
WHERE email NOT LIKE '%@seed.local'

ORDER BY count DESC;
```

**Skills 统计：**
```sql
SELECT
    category,
    type,
    status,
    COUNT(*) as count,
    AVG(rating_avg) as avg_rating,
    SUM(download_count) as total_downloads
FROM skills
WHERE skill_id NOT LIKE 'seed_%'
GROUP BY category, type, status
ORDER BY count DESC;
```

**最近注册用户：**
```sql

    username,
    email,
    role,
    created_at
FROM users
WHERE email NOT LIKE '%@seed.local'
ORDER BY created_at DESC
LIMIT 10;
```

### 8.2 数据完整性检查

**检查外键约束：**
```sql
SELECT
    tc.table_name,
    kcu.column_name,
    ccu.table_name AS foreign_table_name,
    ccu.column_name AS foreign_column_name
FROM information_schema.table_constraints AS tc
JOIN information_schema.key_column_usage AS kcu
    ON tc.constraint_name = kcu.constraint_name
JOIN information_schema.constraint_column_usage AS ccu
    ON ccu.constraint_name = tc.constraint_name
WHERE tc.constraint_type = 'FOREIGN KEY'
ORDER BY tc.table_name;
```

**检查索引使用情况：**
```sql
SELECT
    schemaname,
    tablename,
    indexname,
    idx_scan as index_scans,
    pg_size_pretty(pg_relation_size(indexrelid)) as index_size
FROM pg_stat_user_indexes
WHERE schemaname = 'public'
ORDER BY idx_scan ASC;
```

```sql

    pid,
    usename,
    application_name,
    client_addr,
    state,
    query_start,
    state_change
```

```sql

    pid,
```
